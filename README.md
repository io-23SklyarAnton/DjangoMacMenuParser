# Django Web App for Parsing McDonald's Website and Displaying Menu Info
This Django web application is designed to parse McDonald's website to extract menu information and then display it to users.

## Features
Menu Parsing: The application parses McDonald's website to extract menu information such as product names, descriptions, and nutritional values.

Display Menu: After parsing, the menu information is displayed on the web interface for users to browse.
